# retail-sales-analysis-tableau-sql
An end‑to‑end sales analytics project using SQL and Tableau for business insight generation

## Project Overview
This project presents an end-to-end analysis of retail sales data to understand overall business performance, revenue trends, product contribution, and regional sales distribution.  
The analysis was conducted using SQL for data preparation and Tableau for data visualization.

## Business Objectives
The goal of this project was to answer key business questions such as:
- How is total sales and profitability performing over time?
- Which products contribute most to overall revenue?
- How does sales performance vary across regions?
- What are the high-level trends decision-makers should focus on?

## Tools & Technologies
- **PostgreSQL** – data storage and SQL-based analysis
- **SQL** – data cleaning, transformation, and aggregation
- **Tableau** – interactive dashboard for visualization and insights
- **GitHub** – version control and project documentation

## Data Description
The dataset contains retail sales transactions from 2023 to 2026, including:
- Order-level sales and profit information
- Product categories and sub-categories
- Customer segments
- Regional and geographic sales data

The cleaned dataset was prepared in PostgreSQL before analysis.

## Analysis Approach
1. Cleaned and validated raw sales data using SQL
2. Created derived metrics such as total sales, total profit, and order counts
3. Aggregated sales metrics by month, product, and region
4. Designed a focused executive-level dashboard in Tableau

## Dashboard Overview
The Tableau dashboard highlights:
- Key performance indicators (Total Sales, Total Profit, Total Orders)
- Monthly revenue trends
- Top 10 products by revenue
- Regional sales distribution

## Key Insights
- Sales show clear seasonal variation across months, with strong upward trends towards the end of the year
- A small number of products generate a significant share of total revenue
- Certain regions consistently outperform others, indicating strong geographic demand concentration
